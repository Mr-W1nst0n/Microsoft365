# Collect Hardware IDs for AutoPilot provisioning
On Windows 10/11 Devices
Open at Boot the command prompt (Shift + F10)
Run 'explorer'
Reach the script path
Execute the cmd as Administrator

## Execution
```ruby
'GetAutoPilot.cmd'
```
